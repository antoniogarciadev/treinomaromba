# 🏋️‍♂️ Treino Maromba - Website Fitness Moderno

Este é um projeto de **Landing Page** desenvolvido para o setor fitness (ginásios, estúdios de personal training ou boxes de crossfit). O objetivo foi criar uma interface rápida, visualmente impactante e totalmente otimizada para dispositivos móveis.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído com o que há de mais moderno no ecossistema Frontend:

* **[Vue.js 3](https://vuejs.org/)** (Composition API) - Framework progressivo.
* **[Tailwind CSS](https://tailwindcss.com/)** - Estilização baseada em utilitários para rapidez e consistência.
* **[Swiper.js](https://swiperjs.com/)** - Implementação de sliders dinâmicos com suporte a toque.
* **[Vite](https://vitejs.dev/)** - Ferramenta de build ultra-rápida.

---

## ✨ Funcionalidades Principais

* ✅ **Hero Slider:** Banner principal rotativo com autoplay (Swiper.js) para promoções.
* ✅ **Design Responsivo:** Adaptado para Smartphones, Tablets e Desktop (Mobile-first).
* ✅ **Grades de Treino:** Seção dedicada para mostrar modalidades (Musculação, Yoga, HIIT).
* ✅ **Tabelas de Preços:** Componentes limpos para exibição de planos e benefícios.
* ✅ **Performance:** Código otimizado para carregamento instantâneo.
